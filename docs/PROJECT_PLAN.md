\# PROJECT\_PLAN.md — CareerFit AI 프로젝트 기획 초안



\## 1. 문제 정의



취업을 준비하는 대학생은 수많은 채용 공고와 공모전 중에서 자신에게 맞는 정보를 찾기 어렵다. 또한 자신이 가진 역량이 관심 직무와 얼마나 잘 맞는지, 앞으로 무엇을 준비해야 하는지 판단하기도 쉽지 않다.



CareerFit AI는 사용자의 전공과 보유 스킬을 분석하여 적합한 취업 공고와 공모전을 찾아주고, 부족한 역량까지 알려주는 \*\*취업 준비 나침반\*\* 역할을 한다.



\## 2. 사용자 정의



\* 대상: 취업을 준비하는 대학교 3\~4학년 학생

\* 사용자 고민:



&#x20; \* 어떤 직무가 나와 잘 맞는지 모르겠다.

&#x20; \* 취업을 위해 무엇을 더 준비해야 할지 모르겠다.

&#x20; \* 나에게 도움이 되는 공모전이나 경험을 찾기 어렵다.



\### 사용 시나리오



1\. 사용자가 전공, 보유 스킬, 관심 직무를 입력한다.

2\. CareerFit AI가 관련 취업 공고와 공모전 데이터를 검색한다.

3\. 사용자의 강점과 부족한 역량을 분석한다.

4\. 추천 결과와 판단에 활용한 출처를 함께 제공한다.



\## 3. 데이터 계획



\* 취업 공고 데이터: 기업명, 직무명, 요구 스킬, 우대 전공, 근무 지역, 마감일, 출처

\* 공모전 데이터: 공모전명, 분야, 참가 대상, 필요 역량, 마감일, 출처

\* 데이터 출처: 강사 제공 목업 CSV



\## 4. 기능 목록



| 기능         | 설명                      |

| ---------- | ----------------------- |

| 사용자 정보 입력  | 전공, 보유 스킬, 관심 직무 입력     |

| 맞춤형 데이터 검색 | 사용자와 관련된 취업 공고 및 공모전 검색 |

| 역량 매칭      | 사용자 역량과 요구 역량 비교        |

| 준비 방향 제안   | 부족한 스킬과 추천 활동 안내        |

| 출처 표시      | 분석에 사용한 데이터의 출처 제공      |



\## 5. 향후 개선 아이디어



현재는 핵심적인 역량 분석과 추천 기능에 집중한다.



\* 이력서 PDF 자동 분석

\* GitHub 활동 기반 역량 분석

\* 공모전 및 채용 마감 알림

\* 취업 준비 과정을 보여주는 성장 지도

\* 비슷한 진로를 준비하는 학생들의 익명 역량 비교
